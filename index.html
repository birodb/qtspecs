QT4 CG Homepage

This site is used to publish specifications, meeting logistics, and other information about the QT4† CG (AKA the “XSLT Extensions Community Group”).

The CG has weekly teleconferences on Tuesdays. The meeting logistics page provides additional details. The agendas and minutes are also posted.

Specifications

The latest drafts of

  • XSLT Transformations (XSLT) Version 4.0 (latest diffs)
  • XPath and XQuery Functions and Operators 4.0 (latest diffs)
  • XML Path Language (XPath) 4.0 (latest diffs), and
  • XQuery 4.0: An XML Query Language (latest diffs)

are updated automatically when changes are accepted by the CG.
